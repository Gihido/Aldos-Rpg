# Aldos RPG - Веб-версия

Это улучшенная веб-версия текстовой RPG игры Aldos RPG, адаптированная для запуска в браузере с современным интерфейсом и расширенным функционалом.

## Как запустить игру

### Через VS Code с расширением Go Live Server:

1. Убедитесь, что у вас установлен VS Code и расширение **Live Server** (от Ritwick Dey)
2. Откройте папку `/web-game` в VS Code
3. Найдите файл `index.html` в проводнике
4. Кликните правой кнопкой мыши по файлу `index.html`
5. Выберите "Open with Live Server" или нажмите на значок Go Live в правом нижнем углу VS Code
6. Игра автоматически откроется в вашем браузере

### Вручную через локальный сервер:

Если у вас нет VS Code или расширения Go Live:

1. Перейдите в директорию `/web-game`
2. Запустите локальный HTTP-сервер, например:
   - Python 3: `python -m http.server 8000`
   - Node.js: `npx serve`
   - PHP: `php -S localhost:8000`
3. Откройте в браузере `http://localhost:8000`

## Особенности

- **Улучшенный визуальный дизайн**: Современный интерфейс с анимациями, градиентами и эффектами
- **Полноценная RPG система** с классами персонажей
- **Улучшенная боевая система** с несколькими типами атак (физическая, магическая, защита, использование предметов)
- **Система уровней и опыта** с визуальным отображением прогресса
- **Система инвентаря** с возможностью использования предметов во время боя
- **Система экипировки** с различными типами предметов
- **Сохранение прогресса** в localStorage
- **Аутентификация пользователей** с регистрацией и входом
- **Несколько игровых локаций** с уникальным оформлением
- **Визуальные эффекты** урона, анимации и уведомления
- **Система побега** из боя с определённым шансом на успех
- **Расходные предметы** для восстановления здоровья и маны

## Новые функции

- **Магические атаки**: Использование маны для нанесения дополнительного урона
- **Анимации урона**: Визуальное отображение нанесенного урона во время боя
- **Интерактивные элементы**: Анимации при наведении, эффекты при взаимодействии
- **Улучшенный интерфейс боя**: Более информативное отображение характеристик
- **Система уведомлений**: Анимированные сообщения с различными типами
- **Адаптивный дизайн**: Поддержка различных размеров экранов
- **Улучшенная система предметов**: Различные типы предметов с уникальными свойствами
- **Шкалы HP/MP**: Визуальное отображение уровня здоровья и маны
- **Возможность использования предметов во время боя**: Выбор из инвентаря в бою
- **Улучшенная система защиты**: Уменьшение урона при использовании защиты
