# Запуск игры через VS Code Go Live Server

## Подготовка к запуску

Веб-версия игры Aldos RPG полностью готова к запуску через расширение **Go Live Server** в VS Code.

## Инструкция по запуску:

1. Убедитесь, что у вас установлено расширение **Live Server** от Ritwick Dey в VS Code
2. Откройте директорию `/web-game` как рабочую область в VS Code
3. Найдите файл `index.html` в корневой директории проекта
4. Щелкните правой кнопкой мыши по файлу `index.html`
5. Выберите пункт "Open with Live Server" или нажмите на значок Go Live в правом нижнем углу VS Code
6. Игра автоматически откроется в вашем браузере по адресу вроде `http://127.0.0.1:5500`

## Альтернативный способ запуска:

Если вы хотите запустить игру вручную:
```bash
cd /workspace/web-game
python3 -m http.server 8000
```
Затем откройте в браузере: http://localhost:8000

## Особенности игры:

- Все данные сохраняются в localStorage вашего браузера
- Поддерживается регистрация и вход пользователей
- Реализована боевая система с монстрами
- Доступна система уровней и опыта
- Есть возможность перемещения между локациями

## Структура проекта:

- `index.html` - главная страница игры
- `css/style.css` - стили оформления
- `js/` - скрипты игры:
  - `config.js` - конфигурация игры
  - `utils.js` - вспомогательные функции
  - `monster.js` - класс монстра
  - `player.js` - класс игрока
  - `game.js` - основная логика игры
  - `main.js` - точка входа

Игра готова к запуску через VS Code Go Live Server!
